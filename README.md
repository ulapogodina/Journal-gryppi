# Journal gryppi
Данное консольное приложение реализует базовый учет успеваемости студентов с использованием языка программирования Go. Основной функционал включает добавление студентов, отображение списка, фильтрацию по среднему баллу и подсчет среднего балла для каждого студента.

Возможности
Ввод информации о студенте (ФИО и оценки)
Вывод всех студентов с их оценками и средним баллом
Фильтрация студентов по среднему баллу (например, вывести студентов, у которых средний балл ниже указанного порога)
Подсчет среднего балла для каждого студента
Как пользоваться
Запустите программу.
В меню выберите нужное действие:
Добавить студента — введите ФИО и оценки (через запятую)
Просмотреть всех студентов
Отфильтровать студентов по среднему баллу
Выйти из программы
Следуйте инструкциям, отображаемым в консоли.
Требования
Go (версия 1.18 или выше)
Запуск
Склонируйте репозиторий или скачайте исходный код.
Откройте терминал и перейдите в папку проекта.
Выполните команду:
CopyRun
go run main.go
Структура кода
Структура Student содержит поля:
FIO — ФИО студента
Grades — срез оценок
Average — средний балл
Методы:
CalculateAverage() — рассчитывает средний балл по оценкам
